Substructure¶

Featurizers describing the structure of (and/or the count and/or presence of substructures in) a molecule.

class FragmentSearchFeaturizer(smarts, names, count=True, preset_name='custom')[source]¶

Bases: AbstractFeaturizer

A featurizer for molecular substructure search via SMARTS.

Initialize class.

Parameters:

  • smarts (Optional[List[str]]) – SMARTS strings that are matched with the molecules. Defaults to None.

  • names (Optional[List[str]]) – Names of the SMARTS strings. If None, the SMARTS strings are used as names. Defaults to None.

  • count (bool) – If set to True, count pattern frequency. Otherwise, only encode presence. Defaults to True.

  • preset_name (str) – Name to give preset of interest. Defaults to custom.

classmethod from_preset(preset, count=True)[source]¶

Generate class instance with atomic numbers of interest based on predefined presets.

Parameters:

  • preset (str) – Preset name of the substructures encoded by the SMARTS strings. Predefined presets can be specified as strings, and can be one of: * heterocyclic, * rings, * amino, * scaffolds, * warheads or * organic. * all

  • count (bool) – If set to True, count pattern frequency.
